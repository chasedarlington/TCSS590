## Introduction
The goal of this project is to develop a reinforcement learning agent able to land a lunar module safely within specific landing zones distributed in uneven terrain. 
This agent should learn through interaction with the environment, determining how much thrust and movement is needed to land. 
In the end we'll show how effectively reinforcement learning can solve decision-making problems in a video-game environment from start to finish.


## Proposal
This research proposal shows how reinforcement learning solves decision making problems with applications for robotics, drone navigation, and autonomous control. 
This project introduces uncertainty such as randomized starting positions, limited fuel and different landing conditions to encourage robust learning. 
It enables the agent to learn optimal landing through trial-and-error, balancing learning behaviors and improving old ones. 
Success will be measured by landing rates and fuel efficiency. 
Key challenges include stable learning and balancing exploration with exploitation. 
This project will be worked on in Python using RL tools such as OpenAI Gym.
